WORKFLOW:
================================================================================
1. Read QUICK_START.md
2. Open project in Godot 4.5
3. Build C# solution (Ctrl+Shift+B)
4. Implement scripts following TODO comments
5. Add sprites/sounds to assets/ folders
6. Assign assets in Godot editor
7. Test and tune!


FILE YOU SHOULD MODIFY:
================================================================================
✓ All files in scripts/        - Implement the TODO functions
✓ Files in assets/             - Add your sprites and sounds
✗ Files in scenes/             - Edit in Godot editor, not text editor
✗ Generated files              - Let Godot/compiler manage these


IMPORTANT NOTES:
================================================================================
- Scene files (.tscn) are text but should be edited in Godot
- Build the project before running (Ctrl+Shift+B)
- All C# classes must be "public partial class"
- Node references must be obtained in _Ready() using GetNode<T>()
- Signals are connected with += operator in C#
